Conceitos e Estruturas Básicas de HTML

Todos os documentos HTML devem começar com uma declaração de tipo de documento: <!DOCTYPE html>.
O documento HTML em si começa com <html> e termina com </html>.
A parte visível do documento HTML está entre <body> e </body>.
A Declaração <!DOCTYPE> representa o tipo de documento e ajuda os navegadores a exibir páginas da web corretamente.
Ele deve aparecer apenas uma vez, no topo da página (antes de qualquer tag HTML).
A declaração <!DOCTYPE> não diferencia maiúsculas de minúsculas.
A declaração <!DOCTYPE> para HTML5 é: <!DOCTYPE html>.

Os parágrafos HTML são definidos com a tag <p>:
Por exemplo:
<p>Este é um parágrafo.</p>
<p>Este é outro parágrafo.</p>

Os links HTML são definidos com a tag <a>:
Exemplo: <a href="https://www.w3schools.com">este é um link</a>
O destino do link é especificado no atributo href.
Atributos são usados para fornecer informações adicionais sobre elementos HTML.

Como visualizar o código-fonte HTML

Você já viu uma página da Web e se perguntou "Ei! Como eles fizeram isso?"
Ver código-fonte HTML:
Clique em CTRL + U em uma página HTML, ou clique com o botão direito na página e selecione "View Page Source".
Isso abrirá uma nova aba contendo o código-fonte HTML da página. Inspecionar um elemento HTML:
Clique com o botão direito em um elemento (ou uma área em branco) e escolha "Inspecionar" para ver de que elementos são feitos (você verá tanto o HTML quanto o CSS).
Você também pode editar o HTML ou CSS na hora no painel Elementos ou Estilos que abre.
